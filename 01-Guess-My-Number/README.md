## ðŸ•¹ Game Rules

- The game generates a random secret number between 1 and 20.
- The player enters a guess.
- Feedback is shown if the guess is:
  - Too high
  - Too low
  - Correct
- The score decreases with every incorrect guess.
- A high score is maintained.
- Clicking **Again** restarts the game with a new number.
